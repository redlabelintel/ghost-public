# Memory File Schema

All files in `memory/` should include YAML frontmatter with these fields:

```yaml
---
date: YYYY-MM-DD
type: [standup | decision | fact | research | error]
tags: [tag1, tag2]
confidence: [high | medium | low]
status: [active | resolved | archived]
---
```

## Type Definitions

- **standup**: Daily operational summary
- **decision**: Choices made with reasoning
- **fact**: Observed events or data
- **research**: External analysis or learnings
- **error**: Issues encountered and fixes applied

## Example

```yaml
---
date: 2026-02-19
type: decision
tags: [config, model, cost]
confidence: high
status: resolved
---

# Ghost Agent Switched to Local Model

**Why:** Auth errors from OpenRouter (HTTP 401)
**Grounding:** $0/day cost strategy (Feb 17 standup)
**Action:** Added local:default profile, patched gateway config
```
