## üîß TROUBLESHOOTING GUIDE

### Problem: Dashboard Shows "OFFLINE" (Grey Dot)
**Cause:** Backend API is down  
**Solution:**
1. **Check API directly:** Go to http://100.76.103.90:3001/api/health
2. **If it loads:** Refresh dashboard page
3. **If it doesn't load:** Backend needs restart (contact admin)
4. **Wait 30 seconds** then check dashboard status again

### Problem: Metrics Show "--" 
**Cause:** API connection loading or failed  
**Solution:**
1. **Wait 30 seconds** for automatic refresh
2. **Check System Intelligence Log** for error messages
3. **If errors persist:** Refresh page to restart connections
4. **Check green status dot** - Should be online for metrics to load

### Problem: Chat Not Responding
**Cause:** Chat API connection issue  
**Solution:**
1. **Check System Intelligence Log** for chat errors
2. **Try typing "hello"** - Basic connectivity test
3. **If no response:** Refresh page to restart chat connection
4. **Verify green status dot** - Backend must be online for chat

### Problem: Buttons Don't Work
**Cause:** JavaScript or connection errors  
**Solution:**
1. **Check browser console** for JavaScript errors (F12 ‚Üí Console)
2. **Refresh the page** - Restart all JavaScript
3. **Try basic button first** - [OPEN] WORKSPACE is safest test
4. **Check System Intelligence Log** for button click confirmations

### Problem: Warren Doesn't Respond
**Cause:** CEO approval workflow timing or connection  
**Solution:**
1. **Wait full 3 seconds** after clicking [CEO] REQUEST APPROVAL
2. **Check Warren's status** - Should change to EVALUATING then STRATEGIC
3. **Look in System Intelligence Log** - Warren's response appears there
4. **Be specific in proposals** - "Hire data analyst" vs "strategic initiative"
